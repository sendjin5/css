배치 및 레이아웃 관련 속성

position, left, right, top, bottom, z-index, float, clear 등



position(위치)

현재 선택된 요소를 어떠한 방식으로 배치할지 위치 결정 방식의 속성

사용 가능한 속성 : static(그대로) | relative(상대적) | absolute(절대적) | fiexed(고정적) | sticky

모든 요소는 부모를 따라가며, 위에서 아래, 왼쪽에서 오른쪽으로 배치된다.

static : 정적 배치이며, position의 기본값으로 별도로 position을 지정하지 않는다. 이는 곳 모든 요소를 static으로 결정된다.


relative : 상대 배치이며, 내부요소나 형제요소와 상대적으로 배치할 경우 지정한다.

=>static, relative 는 margin으로 조절하라, 방향위치값을 주면 전 데이터와 배치가 귀찮아진다. 또한 가로 방향으로 여러 요소를 배치할 경우는 float 속성을 활용하여 배치하도록 한다.


absolute | fiexed : 겹치기가 가능하며, 문서상의 순서상의 역순으로 쌓임(최근 것으로 덮어올려짐), 또한 겹치기 진행시 z-index를 지정하여야한다.

=>top/bottom, left/right로 서로 둘 중 하나만 걸어야한다.(반대쪽 값을 정확히 주지않으면 지정되지않음)

=>fixed는 고정적인 메뉴창으로 많이 쓰인다.

=>absolute는 창의 시작점이 기준으로 되며, 이때 relative를 부모에 걸게되면 기준점이 부모로 잡힌다.


sticky : 갑툭튀, 효과가 고정으로 거의 쓰이지 않는다.



위치값 지정

지정가능한 키워드 : auto | length

auto는 현재 left 속성 값이 지정이 되어 있는 경우 right 속성 값을 변경하려고 할 경우(=클락와이즈) => left를 auto로 하며, right 값을 지정한다.



레이어 설정

z-index : 층수(layer)라 생각하며, 방식으로는 숫자(정수)만 쓰인다. 숫자가 클수록 위층에 배치(기본값은 0이다.)

=>만약, z-index 속성 값을 지정하지 않으면, 먼저 작성된 요소(태그)가 밑(뒷면)으로 배치된다.

\

float - 흐름 속성(떠다니다)

지정 가능한 값 : noen | left | right



clear - 흐름 해제 속성

지정 가능한 값 : noen | left | right | both(float, clear 모두 해제)
