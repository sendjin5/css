css 출력 및 박스 관련 속성

display, visibility, opacity, width, height, overflow, margin, padding, border 등

모든 속성의 값에 initial(초기값), inherit(상속-부모요소에 내용을 그대로 자식요소가 받음)

display : 화면 출력 모드를 지정하는 속성

태그마다 화면 출력 모드가 block, inline, inline-block 등으로 구성

inline 속성 요소: span, a, strong, em, ins 등
특징 : 한 줄 안에 자동 줄바꿈이 될 때까지 배치되며, 크기 지정을 할 수 없어 width, height, line-height, text-indent, text-align, paddign-top, padding-bottom, margin-top, margin-bottom 등의 속성을 지정할 수 없다.

block 속성 욧 : div, h1~h6, p, section, article, header, footer
특징 : 자동으로 다음 줄에 배치가 되며, 크기과 관련된 속성을 지정할 수 있다.

부여 가능한 속성의 값 : inline |block | inline-block | none
부여 가능한 속성 값2 : contents, flex, grid, box, inline-flex, inline-grid, inline-table, list-item, run-in table-caption, table-column, table-row, table-cell, table=header-group, table-group, table-column-group, table-row-group 등

visibility(가시속성)

보일지 보이지 않을지를 나타내는 속성

visible | collapse | hidden(영역은 차지하나 보이지 않을뿐, none은 영역도 제외)

opacity(불투명도속성)

불투명 정도를 나타내는 속성

값 : 0~1 까지의 실수 | 정수로 표시하며, 0은 투명, 1은 불투명
해당 요소의 글자와 모든 배경 이미지, 색상을 포함한 모든 콘텐츠를 지정할 수 있다.

width(너비속성), height(높이속성)

가로(폭=너비)의 크기를 지정하는 속성

auto | length | % |

length는 px, pt, pc, cm, mm, in(ch), vw, vmin(현재기기 최소폭), vmax(현재 기기 최대폭) 등의 단위로 지정이 가능

기본 화면 출력속성이 블록요소 또는 인라인블록요소 등에만 크기지정이 가능함.

overflow(흘러넘침속성)

콘텐츠의 양이 지정한 크기(폭/높이)보다 더 클 경우 지정

auto | visibel | hidden | scroll | clip

세부속성 - overflow-x(가로), overflow-y(세로)

margin(바깥여백)

현재 테두리(콘텐츠) 영역의 바깥 여백으로서 콘텐츠와 콘텐츠 사이의 여백

auto | length | %

세부속성 - margin-left | top | bottom | right (통합속성을 먼저 지정 후 다음에 세부속성을 지정하여야 함)

margin 1, 2 ->상하, 좌우 | maring 1, 2, 3 -> 상, 하, 좌우 | margin 1, 2, 3, 4 -> 상, 우, 하, 좌(시계방향)
auto는 좌우만 걸 수 있다. / maring 25px auto -> 세로는 25px, 가로는 auto

padding(안쪽여백 - 콘텐츠 안에 콘텐츠)

지정한 폭/넓이에 padding과 border 값을 더한 값이 실제 콘텐츠의 크기가 된다.

margin과 같음

border(테두리)

width, style, color

ex) border 1px solid #000;
1px 두께의 검은색 실선

width는 선의 두께로 px, pt, pc, cm, mm, in(ch), vw, vmin(현재기기 최소폭), vmax(현재 기기 최대폭) 등의 단위로 지정이 가능

style는 선의 모양으로 solid(실선), none, hidden, dotted(둥근점선), dashed(사각점선), double(두 줄), groove(은각), ridge, inset, outset 등의 지정이 가능

color는 선의 색으로서 컬러명, 16진(hex)코드, rgb()/rgba(), hsl()/hsla() 등으로 지정이 가능

카테고리별 세부속성

border-width(선두께) | style(선종류) | color(선색상)

margin과 동일한 지정방식이며, (상우하좌) 지정 따로 따로 가능

방향별 세부속성

border-left(왼쪽테두리) : 2px double, gold

세세속성(예시)

border-left-width | border-right-style 등 가능

border-방향-속성 순

box-sizing(박스의 크기 산정방법)

박스의 크기를 지정한 너비/높이의 콘텐츠, padding, border의 포함유무를 지정

content-box(콘텐츠만 지정) | border-box(콘텐츠+padding+border로 적용)

border-radius(테두리둥글기)

선의 두께로 px, pt, pc, cm, mm, in(ch), vw, vmin(현재기기 최소폭), vmax(현재 기기 최대폭) 등의 단위로 지정이 가능

ex1) border-radius: 20px

ex2) border-radius: 50%

ex3) border-radius: 상, 우, 하, 좌 가능

ex4) border-top-left-radius: 20px (순서: 위좌, 위우, 아래우, 아래좌)
